## **～项目概述～**

「秋叶圆体 16」是基于「KHドット秋葉原16」增补简繁体汉字的16px点阵圆体，适合在各种像素风场景使用。

中文像素字体长期缺少圆体风格的字体，本补字项目为填补此空白而诞生。

于是，在经过将近两个月的调整和补字后，本字体终于发布。

与原字体相比，本字体修复了错误的字形，为保持视觉平衡略微扩大了中宫，且尽可能倒圆了包括竖提在内所有边角，仅剩下部分角度过小难以倒圆的锐角维持了原设计。

希望这款字体能为像素艺术创作和游戏本地化工作添砖加瓦。

目前本字体项目尚未完工，且仍在持续更新，欢迎各位网友提出宝贵意见。
